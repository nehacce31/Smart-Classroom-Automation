# Smart Classroom Automation using ESP32

## Description
A Smart Classroom Automation System developed using ESP32 to automatically control classroom lighting based on occupancy and ambient light conditions.

## Features
- Automatic light control
- Occupancy detection
- NeoPixel LED status indication
- Buzzer alert for energy saving

## Components Used
- ESP32
- LDR Sensor
- IR Sensor
- NeoPixel Ring
- Buzzer
